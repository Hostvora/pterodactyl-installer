# Hostvora Pterodactyl Installer

Ein interner Installer für die automatische Einrichtung von **Pterodactyl Panel** und **Wings** auf unseren Hostvora-Servern.

## Funktionen

* Installation des Pterodactyl Panels
* Installation von Pterodactyl Wings
* Automatische Installation aller benötigten Abhängigkeiten
* Einrichtung von Datenbank, Webserver und Systemdiensten
* Optionales SSL-Setup
* Optionales Firewall-Setup
* Deinstallation von Panel und Wings

## Unterstützte Systeme

* Ubuntu 22.04+
* Debian 12+
* AlmaLinux 9
* Rocky Linux 9

## Hinweise

* Die Installation sollte auf einem frischen Server erfolgen.
* Root-Zugriff wird vorausgesetzt.
* Docker wird bei der Wings-Installation automatisch eingerichtet.
* SSL und Firewall können während der Installation optional aktiviert werden.

## Intern

Dieses Projekt ist ausschließlich für den internen Einsatz bei **Hostvora** vorgesehen und wird nicht öffentlich unterstützt.
